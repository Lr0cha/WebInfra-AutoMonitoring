Infraestrutura Web com Monitoramento Automatizado

Etapa 1: Configuração do Ambiente

Preparação de um ambiente local Linux com sistema próprio ou WSL.

Etapa 2: Servidor Web

  • Instalação e configuração do Nginx.
  • Criação de uma página HTML para exibição no servidor.

Etapa 3: Monitoramento Automatizado

  • Script verifica a cada minuto se o site está online.
  • Notificação via Telegram ou Discord em caso de falha.
  • Logs armazenados em /var/log/meu_script.log.

Etapa 4: Testes e Documentação

Desafio Bônus

Automatização completa via User Data para provisionamento inicial.
